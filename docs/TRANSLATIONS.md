## 如何贡献新翻译

1. 复制 `README.en.md` 为 `README.<lang-code>.md`
2. 翻译内容，保持 Markdown 结构不变
3. 更新文件顶部的语言切换链接
4. 在所有已有 README 的语言切换行中添加您的链接
5. 提交 Pull Request

语言代码遵循 ISO 639-1 标准（如 `zh`、`en`、`ja`、`ko`、`es` 等）。

## 规范

- 翻译内容而非代码：保持代码块、命令、路径为英文
- 保持 Markdown 结构和锚点链接不变
- 如果某个概念在目标语言中没有对应术语，可以保留英文并加注
